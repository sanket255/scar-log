# scar-log

A public dev diary documenting a self-directed Python learning journey — one day at a time.

## What this is

Daily entries written in a notebook-style format. Each one covers what was built, what broke, what clicked, and what didn't. No highlight reels. No skipping the embarrassing bugs.

Started from zero (literally `2 + 2`) and working toward building real projects — APIs, file systems, data structures, and eventually something worth showing people.

## What it looks like

Notebook aesthetic. Dark mode. Spiral holes. Washi tape. The works.
Built with plain HTML and CSS — no frameworks, no build tools, no npm install.
Hosted on GitHub Pages.

**Live:** [sanket255.github.io/scar-log](https://sanket255.github.io/scar-log)

## Structure

```
/
├── index.html        # Homepage listing all entries
├── shared.css        # Single stylesheet for everything
├── day-0/
│   └── index.html
├── day-1/
│   └── index.html
└── ...
```

## Stack

- HTML + CSS + vanilla JS
- GitHub Pages for hosting
- No dependencies

## Why public

Accountability. Proof of work. And because the part nobody posts —
the bugs, the confusion, the 25 minutes staring at one line —
is actually the most useful part to read.
