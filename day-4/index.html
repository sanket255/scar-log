Day 4
Running on Fumes
a dev diary by scar  /  episode 4 of ∞

Quick life update: I took a whole week off from coding. Not planned. Not strategic. Just happened. Like how a plant dies — not because you wanted to kill it, but because you kept meaning to water it tomorrow.

today's battery level: critically low

Came back tonight at roughly 70% functional human. Exhausted from life, behind on the challenge, acutely aware that my uncle is somewhere waiting for proof that this wasn't a waste of time.

GitHub needs green squares. Uncle needs evidence. I need sleep. Only two of these things happened tonight. I'll let you guess which one didn't.

Built a password checker. On fumes. At 11PM. Partly for learning. Partly for the GitHub commit graph. Entirely because "I'll do it tomorrow" has a 0% success rate in my hands.
bug #1: the boolean incident

Wrote if user_pw.isdigit() == "true": and then stared at it for thirty minutes wondering why Python was personally victimizing me.

Turns out .isdigit() returns the boolean True — not the string "true". No quotes. Capital T. Completely different thing. Python is very particular about this. Python does not care that you're tired. Python has no empathy.

1 / 3
bug #2: the loop that wouldn't shut up

Wrote a loop to check if the password has a digit. It printed "found digit" for every single digit in the password. Password has 4 numbers? Congrats, you get 4 announcements. Very enthusiastic. Not useful.

Learned about break. break tells the loop: we found it, you can stop performing now. The catch — break has to be inside the if, not outside it. One indent. Completely different behaviour. Python once again: no notes, no mercy.

bug #3 & #4: the counter that had amnesia

Created strength = 0 after the loop that needed it. Tried to increment something that didn't exist yet. Smooth. Moved it before. Fine.

Then the counter never reset between passwords — so password #2's score was just vibing on top of password #1's score. The fix: reset at the start of each loop, not once at the top. Order matters. Tremendously. Unforgivingly. Always.

4 bugs. 4 fixes. ~2 hours. ~30 lines. The password checker now takes input, checks for digits, uppercase, and special characters, rates strength, and loops until you quit. It works. I tested it. It works. That sentence felt very good to type.
