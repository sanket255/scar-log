Day 7
Functions Meet Files
a dev diary by scar  /  episode 7 of ∞

Built something today that looked simple on paper. Which, historically, is how I know it's about to go wrong.

what i thought i was building

A note manager. Write notes. Read notes. Loop until you quit. Functions plus files — two things I already "understood." Which means, naturally, I made it way more complicated than it needed to be.

the mess (self-inflicted)

Started with one variable doing everything. date was handling input, decisions, life choices — all of it. Conditions were checking the wrong thing. Flow made no sense. The code technically existed, but it wasn't thinking.

This is a pattern now: I don't fail because something is hard. I fail because I try to be clever before being clear.
the fix (boring, effective)

One variable. Clear options. No overthinking. Ask the user what they want, call the function, loop. That's it. That was the whole solution. Not new knowledge. Just cleaner thinking.

1 / 3
the bug that always worked (worst kind)
if ask == "s" or "w":

This line worked. Which was the problem. It always worked.

Turns out "w" is always truthy in Python. So the condition was basically: "if anything happens at all, do this." Every branch. Every time. No exceptions. Debugging something that technically runs but is completely wrong is a special kind of pain I wouldn't wish on anyone.

Fix: if ask == "s" or ask == "w": One extra comparison. Completely different logic. Python requires you to be specific. It will not infer what you meant. It does not know you, and it does not care.

what combining concepts actually means

Functions handle one thing. Files remember things. Main decides what happens. That's the system. Not complicated — just easy to break if you don't respect each part's job. Today I mostly respected it. On the fourth attempt.

the actual realization

Combining concepts isn't harder than learning them separately. It just exposes whether you actually understood them — or just understood them in isolation.

Today I found out the difference firsthand. Took four attempts. Got there.

That's enough.
