Day 8
The Journal Lives
a dev diary by scar  /  episode 8 of ∞

Built a real app today. Not practice. Not a “toy project.” Something I could actually use.

It saves. It reads. It searches. It works.

Finished it in 32 minutes. Which feels suspicious, considering yesterday existed.

what changed

Yesterday: 2 hours fighting main(). Today: wrote it in minutes.

Same idea. Same structure. Just… no resistance this time.

This is new: The code didn’t fight back.

Which probably means I’m finally starting to understand what I’m doing.
1 / 3
the structure (kept it boring)

One loop. One question. Four options: save, read, search, quit.

No extra logic. No clever tricks. Just:

ask → decide → call function → repeat

That’s the whole app. Which is slightly annoying, considering how complicated I made this yesterday.

the search thing

Didn’t overthink it.

Loop through the file. Check if the keyword exists. Print the line.

That’s it. No fancy structures. No “smart” solutions. Just the obvious one.

Turns out “simple” only works after you stop trying to be impressive.
2 / 3
the actual realization

Yesterday I was figuring out how things work. Today I just… used them.

Same concepts. Different feeling.

Repetition doesn’t make things easier. It makes them obvious.

And once something feels obvious, you stop fighting it.

That’s when things start to move.

— scar
